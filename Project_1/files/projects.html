Projects

Colour Changing lamp Using Sensors in Multisim

The Project was aimed at building a Colour Changing Lamp using Sensors, which can be used in many diverse areas like Partying, House decoration, Alarm alerts. It mainly uses Resistive touch sensors and Multisim development platform to create multiple colours from people present in respective region.

Petri Net Model for Traffic-Responsive Signalling Control

The Project was aimed at building a Petri Model for traffic control which focus at minimizing queue lengths by optimizing duration of each signal. It mainly uses Deterministic and Stochastic Petri Net(DSPN) model for the oversaturated traffic.
